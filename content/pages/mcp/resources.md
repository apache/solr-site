## Tutorials ##

* [Solr MCP Server Quick Start](https://github.com/apache/solr-mcp#get-started-users)
  Get from zero to a working AI assistant + Solr integration in under 2 minutes using Docker Compose and Claude Desktop.

* [Solr MCP Server User Guide](https://solr.apache.org/guide/solr/latest/getting-started/tutorial-solr-mcp.html)
  The comprehensive reference guide tutorial covering STDIO and HTTP transport modes, MCP client integration (Claude Desktop, Claude Code, VS Code, Cursor, JetBrains), OAuth2 security, observability with the LGTM stack, and building from source.
