Apache Solr Operator™ v0.9.1

Solr Operator Features

The Solr Operator is a control-plane manager for a Solr ecosystem in Kubernetes.

  • Solr Cloud

    Orchestrate highly available Solr Cloud clusters in Kubernetes.

  • Solr Prometheus Exporter

    Manage collection of Solr Metrics, adapting to changes in your Solr Cloud.

  • Solr Backup

    Take backups of your Solr Cloud.
Solr Cloud

Managing Solr Cloud clusters in Kubernetes.

  • Advanced and Safe Rolling Restarts and Upgrades

    The Solr Operator can orchestrate a safe rolling restart and upgrade of SolrCloud clusters.

  • Easy enablement of Solr Security

    Turning on Solr security, with Basic Auth, is as easy as flipping a switch.

    It's also never been easier to enable SSL within your Solr Cloud.

  • Hassle-Free ZooKeeper Deployment

    Easily spin up an external ZooKeeper cluster for your SolrCloud. Possible via an integration with the ZooKeeper Operator.

  • Easy Addressability for any Provider

    With complete customizable addressability, you can run and expose your Solr Nodes in any Kubernetes Provider. Setup is automatic, all you have to do is provide the settings that work in your environment.

  • Customize the Kubernetes Components

    The Solr Operator gives many options for customizing the Kubernetes components used to deploy the Solr Cloud.
Solr Prometheus Exporter

Managing Solr Metrics in Kubernetes.

  • Automatically Update Information

    Automatically restart when updates to TLS secrets, Auth or ZooKeeper Connection Strings are made.

  • Integrate with Prometheus Stack

    Integration with the Prometheus Operator in Kubernetes is easy and well documented.

  • Customize the Kubernetes Components

    The Solr Operator gives many options for customizing the Kubernetes components used to deploy the Solr Prometheus Exporter.
Solr Backups

Backup your SolrClouds in Kubernetes.

  • Multiple Storage Options

    You have the ability to store your backups in S3 or Kuberenetes Persistent Volumes.
